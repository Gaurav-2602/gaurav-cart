# 🛒 GauravCart

A modern and responsive **e-commerce web application** built using React and Vite.
It allows users to browse products, filter categories, search items, and manage a dynamic shopping cart seamlessly.

---

## 🚀 Live Demo

👉 https://gaurav-cart.vercel.app

---

## 📌 Features

* 🔍 Real-time product search
* 🏷️ Category-based filtering (Laptop, Phone, Camera, Tablet)
* 🛒 Add to cart functionality
* ➕ Increase/decrease product quantity
* 💰 Dynamic cart price calculation
* 📱 Fully responsive design

---

## 🛠️ Tech Stack

* ⚛️ React
* ⚡ Vite
* 🎯 Context API (State Management)
* 🎨 CSS

---

## 📂 Project Structure

```
src/
 ├── components/   # UI components (Navbar, ProductCard, etc.)
 ├── context/      # Global state (Cart management)
 ├── data/         # Product data
 ├── pages/        # Application pages
 ├── App.jsx
 └── main.jsx
```

---

## ⚙️ Installation & Setup

```bash
git clone https://github.com/Gaurav-2602/gaurav-cart.git
cd gaurav-cart
npm install
npm run dev
```

---

## 📦 Build for Production

```bash
npm run build
```
